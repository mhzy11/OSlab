5. SW64 vs RISC-V 中断机制对比

flowchart LR subgraph riscv["RISC-V"] direction TB R1["CPU 硬件"] -->|"查 stvec CSR"| R2["__alltraps"] R2 --> R3["SAVE_ALL"] R3 --> R4["trap()"] R4 --> R5["sret 返回"] end subgraph sw64["SW64"] direction TB S1["CPU 硬件"] -->|"中断信号"| S2["hmcode"] S2 -->|"查 vcpucb"| S3["entInt / entIF"] S3 --> S4["SAVE_ALL"] S4 --> S5["trap()"] S5 --> S6["HMC_rti 返回"] end style S2 fill:#ffe0e0

核心差异：SW64 多了一层 hmcode（红色），它负责保存部分上下文、区分中断类型、查找内核入口。
内核必须通过 trap_init() 提前把 entInt/entIF 地址写入 hmcode 的 vcpucb。
